# Quick start

```r
library("remotes")
install_github("coderaanalytics/econdatar")
```

Install from disk

```r
install.packages("path/to/econdatar",repos=NULL,type="source")
```

Please see the [EconData blog](https://randomsample.co.za) for in depth tutorials
